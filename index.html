Projects

Chip8 Emulator: I created an emulator for Chip8 in order to learn more about system architecture, working of CPU and basics of emulator development. The emulator is interpretative which means it reads the instructions written in the rom file one by one and performs necessary action as defined in the Chip8 documentation. The supporting hardware is includes a display, keyboard, timer and sound. Everything except sound has been implemented as of now. The emulator is written in C++ and Simple DirectMedia Layer (SDL2) is used to implement supporting hardware. I have documented almost all the Chip8 opcodes/instructions in the CPU source file which you can find under src/chip8.cpp in the repo linked below.

Repo link: https://github.com/aayushsrivastava/CHIP-8-Emulator

Image Viewer: A very simple image viewer/gallery application made with Swing in Java. Created for a college programming course.

Find nearby restaurant: This was a twitter bot that I made in python for a hackathon in my first year. The bot used zomato's API and tweepy (wrapper for twitter api). If you tagged the bot's username in a tweet along with a location, the bot would search for a suitable restaurant in the location mentioned and tweet you back with a link to the restaurant.
